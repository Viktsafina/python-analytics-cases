# Python Analytics Cases

Репозиторий содержит аналитические кейсы, выполненные с использованием Python.  
Основной фокус — анализ пользовательского поведения, активности клиентов и покупательских паттернов на основе табличных данных.

## Используемые инструменты
- Python
- pandas
- numpy
- matplotlib
- Jupyter Notebook


## Кейс 1. Анализ клиентской базы фитнес-сервиса

**Файл:** `python_fitness_client_analysis.ipynb`

В рамках кейса проанализировано поведение клиентов фитнес-сервиса с целью:
- выявить ключевые группы пользователей;
- оценить различия в активности и формате тренировок;
- проанализировать динамику клиентской базы во времени.

### Основные этапы анализа:
- предварительная обработка и очистка данных;
- анализ клиентов с преобладанием индивидуальных тренировок;
- выделение наиболее активных пользователей;
- анализ изменения активности и числа клиентов по месяцам;
- интерпретация результатов и формулирование выводов.


## Кейс 2. Анализ активности и покупательского поведения клиентов

**Файл:** `python_client_activity_analysis.ipynb`

Кейс посвящён анализу пользовательской активности и покупок клиентов сервиса.

### В рамках анализа:
- изучается распределение активности пользователей;
- анализируются транзакции и вклад разных групп клиентов в общие показатели;
- выполняется сегментация клиентов по числовым признакам с использованием интервального разбиения (binning);
- сравниваются поведенческие характеристики полученных групп.


## Данные

В обоих кейсах используются обезличенные учебные данные, включённые в репозиторий для воспроизводимости анализа.
